# Macrayola

## Crochet y Macramé Artesanal

Piezas únicas hechas a mano con amor. Descubre nuestros productos artesanales y aprende en nuestros cursos presenciales.

## Tecnologías

Este proyecto está construido con:

- React
- Vite
- TypeScript
- Tailwind CSS
- Firebase (Authentication & Firestore)
- Lucide React (Iconos)

## Instalación y Desarrollo

1. Clona el repositorio.
2. Instala las dependencias:
   ```sh
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```sh
   npm run dev
   ```

## Despliegue

Para construir el proyecto para producción:
```sh
npm run build
```
